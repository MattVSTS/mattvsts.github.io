--- layout: post title: A clarification on the Team Foundation Server Integration Platform support date: '2013-04-14T15:41:00.003+01:00' author: Matteo Emili tags: modified_time: '2013-04-14T15:41:30.011+01:00' thumbnail: http://lh3.ggpht.com/-GPIaLnSucHo/UVs4RR5ejRI/AAAAAAAAA1A/robvZYpEw8s/s72-c/wlEmoticon-smile2.png?imgmax=800 blogger_id: tag:blogger.com,1999:blog-9050180307874313601.post-3548697907410131639 blogger_orig_url: https://mattvsts.blogspot.com/2013/04/a-clarification-on-team-foundation.html --- Often many people think that as the Team Foundation Server Integration Platform is an open-source project, it has no support.

WRONG!

It is the only Microsoft supported solution for all the migration needs you might have.
But only if you download the Visual Studio Gallery package, which is the stable and supported version. If you have an issue with it, you can raise a call to the Customer Support Service.

If you download the Codeplex package (with the source code), you might get the betas, which are obviously not supported.

I hope this clarifies a bit around this really important topic
